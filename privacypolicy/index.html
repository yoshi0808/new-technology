プライバシーポリシー

このプライバシーポリシーは日本国における2022年4月1日より施行の改正個人情報保護法に基づき制定しています。

個人情報および個人関連情報の取り扱い

私はこのサイトの運営において個人情報の収集は行いません。個人関連情報であるクッキーについては収集します。

このサイトを閲覧いただく場合、このプライバシーポリシーをご理解いただき、プライバシーポリシーの事前同意をお願いしております。

私は次に掲げる場合を除いて、あらかじめユーザーの同意を得ることなく、第三者に個人情報および個人関連情報を提供することはありません。ただし、個人情報保護法その他の法令で認められる場合を除きます。

人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき

公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき

国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

このサイトを運営する上で合理的な理由がある場合において、個人情報および個人関連情報の取り扱いについて変更する場合があります。

個人情報および個人関連情報の利用目的

私は個人情報および個人関連情報をサイトの効果的なサイト運営を目的とし利用します。具体的にはクッキーを使い、以下の項目を取得しユーザーが利用しやすいサイト構成や興味のある記事に関連した情報を配信するために利用します。

  • ページ単位のアクセス数
  • ユーザーの属する国、地域
  • ユーザーが利用するOS、ブラウザ、デバイスの種類（PC、モバイル、タブレット）

これらは個人を特定する情報とは関連付けられません。

また、メールなどでご質問、ご要望をいただいた場合に個人情報または個人関連情報が含まれていた場合は、その回答に利用します。

このサイトに関してメールやその他の方法において、情報や質問などを受け取った場合に個人情報が含まれていると考えられる場合は、いただいた情報や質問に回答する目的以外には個人情報は利用しません。また、目的を達成し不要となった個人情報は遅滞なく消去します。

個人関連情報の取得方法

このサイトではプライバシーポリシーに事前同意をいただいた上でクッキーを利用します。クッキーの内容はお使いのブラウザに保存されます。このサイトを訪問するたびにクッキーの内容を読み込む事によって情報を取得します。同意いただけない場合はクッキーへの情報の保存は行いません。

  1. プライバシーポリシーの同意記録
    （目的）プライバシーポリシーに同意いただけたか判定するための記録をします。

  2. Google Analyticsの利用
    （目的）Google Analyticsを利用し、個人を特定する情報とは関連づけられないデバイス情報、アクセス情報を収集します。

GitHub Pagesについて

私は作成したコンテンツを米国GitHub社が提供するGitHub Pagesにアップロードし、ユーザーがコンテンツを閲覧できるようになっています。GitHub社は法的義務を遵守するためならびにウェブサイトおよびサービスのセキュリティおよび統合性を保持するために、GitHub Pagesウェブサイトへの訪問者から、IPアドレスを含むユーザ個人情報を収集することがあります。

「GitHubのプライバシーについての声明」に関しては以下の通りです。
https://docs.github.com/ja/site-policy/privacy-policies/github-general-privacy-statement

なお、GitHubのサービス全般の運営上、プライバシーポリシーにおいてはCookieを利用する事となっていますが、このホスティングサービスである、GitHub Pagesにおいては、GitHubがCookieを埋め込むような事はしないと明言されています。
https://github.community/t/cookies-if-i-use-github-pages-am-i-using-cookies/130249

GitHubで個人情報および個人関連情報がどのように扱われるかはGitHub社の以下の宣言によって確認できます（前述した「GitHubのプライバシーについての声明」に記載されています）。

当社が収集したお客様の情報の共有方法
次のいずれかの場合において、当社はお客様の情報を第三者と共有することがあります。
・お客様の同意がある場合
・当社のサービスプロバイダ間と共有する場合
・セキュリティを目的とする場合
・当社の法的義務を遵守する必要がある場合
・事業法人または事業部門について支配者の変更または売却が行われた場合

当社が個人情報を販売することはありません。GitHubでは広告を掲載することもありません。 お客様の個人情報にアクセスするサービスプロバイダのリストはお客自身で確認できます。

私はGitHub社から個人情報または個人関連情報は受け取りません。また、私はGitHub社が個人情報として使用または認識できる情報をGitHub社に送信したり、第三者によるかかる行為を支援または許可しません。

なお、このサイトはインターネット上に公開されているものであり、このプライバシーポリシーの同意の可否に関わらず、IPアドレスのアクセス記録を技術的に記録しないようにする事はできません。

Google Analyticsについて

このサイトでは、米国Google社が提供するGoogle Analytics4を利用しています。プライバシーポリシーに事前同意いただいた場合のみ、Google Analyticsを用いてアクセス情報の収集を行います。

Google Analyticsでは、Cookie の使用、モバイル デバイスの識別情報（Android の広告識別子、iOS の広告識別子など）、またはデータの収集に使われる類似の技術が用いられます。

このプライバシーポリシーの文末にプライバシーポリシーの非同意への切り替えのリンクがあり、そちらをクリックされた場合は、それ以降のクッキーの書き込みは行わず、アクセス情報の収集は行いません。

Google のサービスを使用するサイトやアプリから収集した情報の Google による使用
https://policies.google.com/technologies/partner-sites

Googleプライバシーポリシー
https://policies.google.com/privacy

このサイトではGoogle社から提供されているgtag.jsというJavaScriptライブラリを使い、クッキーの書き込み可否の制御を行なっています。従い、このサイトのプライバシーポリシーの同意の可否に関わらず、JavaScriptのダウンロードは必要であり、ダウンロード時のIPアドレスのアクセス記録を技術的に記録しないようにする事はできません。プライバシーポリシーの同意をいただいた後は、GoogleAnalyticsにデバイス情報およびアクセス情報を送信しますが、IPアドレスは匿名として送信されます。私はGoogle社から個人情報または個人関連情報は受け取りません。また、私はGoogle社が個人情報として使用または認識できる情報をGoogle社に送信したり、第三者によるかかる行為を支援または許可しません。

Cloudlfare Web Analyticsについて

このサイトでは、米国Cloudlare社が提供するWeb Analyticsを利用しています。Web AnalyticsはIPアドレスを記録せず、またクッキーを利用しません。プライバシーに配慮したAnalyticsです。このサイトではアクセスがあった場合に、Web Analyticsにアクセスの通知を行う信号を出します。Web Analyticsではモバイルやデバイスの識別情報を匿名の形で収集します。

Cloudflare Web Analytics
https://www.cloudflare.com/ja-jp/web-analytics/
Cloudflare Web Analyticsでは、CookieやlocalStorageなどクライアントサイドの状態を一切使わずに、利用に関するメトリクスを収集します。また、分析を表示する目的で、IPアドレス、ユーザーエージェント文字列、その他のデータを介して、個人の「フィンガープリント」を取得することもありません。当社の分析は非侵襲的で、訪問者のプライバシーを尊重します。

Cloudflareは上記ではIPアドレスを取得することはないと上記で宣言していますが、Cloudflareの全てのサービスに関するプライバシーポリシーとしてはアクセスログとしてIPアドレスの利用の可能性に言及しています。私はCloudflare社から個人情報または個人関連情報は受け取りません。また、私はCloudflare社が個人情報として使用または認識できる情報をCloudflare社に送信したり、第三者によるかかる行為を支援または許可しません。

Cloudflare プライバシーポリシー
https://www.cloudflare.com/privacypolicy/
Cloudflareは、エンドユーザーが顧客のインターネットプロパティおよびサービスとのやり取りを行ったログを処理します。この情報は、エンドユーザーが当社のサービスを使用するお客様のドメイン、ウェブサイト、API、アプリケーション、デバイス、エンドポイント、ネットワークにアクセスまたは使用する場合、および Cloudflare for Teams などのサービスにエンドユーザーがアクセスまたは使用する場合に処理されます。処理される情報には、IP アドレス、システム構成情報、およびお客様の Web サイト、デバイス、アプリケーション、ネットワークとの間のトラフィックに関するその他の情報が含まれますが、これらに限定されません。

CDNの利用について

このサイトではオープンソースを活用しています。また、コンテンツデリバリネットワーク（CDN）と呼ばれる、世界中に設置されたサーバーからユーザーに高速にJavaScriptのライブラリを配信する企業があります。このサイトではポーランドのProspect One社が提供する世界中に配置されたjsDelivrサイトからJavaScriptをダウンロードするようになっています。この際、JavaScriptをダウンロードするためのIPアドレスがCDNのサーバー群に記録される場合があります。ダウンロード対象は以下のJavaScriptであり、ユーザーの思考・プライバシーとは無関係です。前述したこのプライバシーポリシーに同意いただくための仕組みには必要なスクリプトであり、このサイトの管理のために技術的に必須です。従い、このサイトのプライバシーポリシーの同意の可否に関わらず、IPアドレスのアクセス記録を技術的に記録しないようにする事はできません。また、jsDelivrのプライバシーポリシーでは、個々のユーザーを追跡しない事が宣言されています。私はProspect One社およびjsDelivrから個人情報または個人関連情報は受け取りません。また、私はProspect One社が個人情報として使用または認識できる情報をProspect One社に送信したり、第三者によるかかる行為を支援または許可しません。

jsDelivr プライバシーポリシー（英語）

JavaScrit
js-cookie

このサイトの安全管理措置について

セキュリティに関しては私が定期的にこのサイトで利用しているソフトウェア群のパッチ情報を入手し最新のソフトウェアに反映するように努めています。私が利用しているオープンシステムは原則としてGitHubにリポジトリがあるオープンなソフトウェアを利用しています。脆弱性がある場合や最新モジュールが提供された場合、それを利用しているユーザーに通知が届くようになっており、速やかにパッチなどが適用できる仕組みになっています。

このサイトではサーバーサイドでデータベースを稼働させていません。情報を閲覧する機能のみを提供しており、ユーザーからの入力を登録・格納できません（検索のために文字を入力できますが、情報の登録はできません）。これは情報漏洩対策、改ざん防止対策に有効です。このサイトのコンテンツを更新できるのは私のみであって、GitHubに対し多要素認証または公開鍵認証を用いた当人認証を行う事で安全面を確保しています。

他サイトへのリンクについて

このサイトでは外部サイトへのリンクが含まれます。これらの外部サイトは私の管理下にはなく、当該サイトのプライバシーポリシーはそれぞれのサイトにおいて確認が必要です。このサイトの記事ではリンク先のURLを可能な限り明示しており、外部サイトとわかるようにしています。情報を共有する目的をもって外部サイトのURLを掲載していますが、当該サイトのポリシーや合法性および安全性については当方は評価しません。訪問するすべてのサイトのプライバシーポリシーを確認することを強くお勧めします。

アフェリエイトリンクは使用しません

このサイトではアフェリエイトリンクをはじめ、広告や寄付などの収入は一切得ていません。また製品・商品の紹介は、具体的な製品の値段をユーザーに知らしめる目的を除き、可能な限り広告掲載が少ないサイトを選定しています。

子供のプライバシー

このサイトは、18歳未満の方（「お子様」）の方には対応していません。
私は、18歳未満の人から故意に個人を特定できる情報を収集することはありません。親または保護者の方で、お客様の子供が何らかの方法によって個人データを提供していることをご存知の場合は、ご連絡ください。親の同意を確認せずに、お子様から個人データを受け取った場合は、その情報を削除する措置を講じます。

プライバシーポリシーの変更について

このサイトは、個人情報および個人関連情報に関して適用される日本の法令を遵守するとともに、本ポリシーの内容を適宜見直しその改善に努めます。修正された最新のプライバシーポリシーは常に本ページにて開示されます。
